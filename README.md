## Description
A responsive homemade food business website developed using HTML, CSS, and JavaScript.  
The project includes user login, dynamic menu display, cart functionality, and an about/contact section.

## Features
- Login authentication using JavaScript and Local Storage
- Responsive menu page with product images
- Add to cart and remove from cart functionality
- Dynamic total price calculation
- About Us and Contact section
- Pastel-themed responsive UI design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage
- Git & GitHub
